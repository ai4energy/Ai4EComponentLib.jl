Ai4EComponentLib.jl

Ai4EComponentLib.jl is a Component library based on the ModelingToolkit acasual modeling system.

Installation

Import Ai4EComponentLib.jl in the standard way:

import Pkg; Pkg.add("Ai4EComponentLib")

Citation

If Ai4EComponentLib is useful, please cite the this paper:
